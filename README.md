#  Smart Note

A modern and intuitive note-taking application built with Flutter and Firebase. Smart Note helps users organize their daily tasks and ideas with categories, pinned notes, and a clean user interface.

---

## Features

*  User Authentication

  * Sign Up
  * Login
  * Secure user sessions

*  Note Management

  * Create notes
  * Edit notes
  * Delete notes

*  Categories

  * Organize notes by category
  * View notes category-wise

*  Pinned Notes

  * Pin important notes
  * Quick access to frequently used notes

*  Recent Notes

  * Display recently created notes

*  Search

  * Find notes quickly

*  Firebase Integration

  * Firebase Authentication
  * Firebase Realtime Database

*  Modern UI

  * Clean and responsive design
  * User-friendly experience

##  Getting Started

### Clone the repository

```
git clone https://github.com/codefriendly-app/Smart-Note.git
```

### Navigate to the project

```
cd Smart-Note
```

### Install dependencies

```
flutter pub get
```

### Run the application

```
flutter run
```

---

##  Firebase Setup

1. Create a Firebase Project.
2. Enable Authentication.
3. Enable Realtime Database.
4. Add Android application.
5. Download `google-services.json`.
6. Place it inside:

```
android/app/
```

7. Run the project.
